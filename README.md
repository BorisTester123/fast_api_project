# FastAPI Books API 
Проект для работы с API: реализован с помощью фреймворка fastAPI, и с настроенным асинхронным подключением и сессией 
к реляционной базе данных PostgreSQL.
Создание таблицы в базе данных и описание с помощью библиотеки sqlalchemy.orm. Также настроен репозиторий для работы с базой, чтобы все запросы уходили именно в базу.

#### **Суть проекта:**

Аутентификация пользователей, базовая авторизация (Basic Auth) с безопасным хранением паролей через bcrypt.
Создание новой таблицы, в которой храним креды авторизации для работы с OpenAPI и CRUD - операциями. 
(Создание, чтение, обновление и удаление книг с базы данных).

**Автотесты**: используется фреймворк Pytest для удобного запуска тестов и проверки API.
**@pytest.mark.parametrize** - декоратор для проверки нескольких сценариев сразу (positive/negative).
Фикстуры для подготовки тестового окружения / данных.

**Стэк**:
* FastAPI
* SQLAlchemy / PostgreSQL
* bcrypt для хеширования паролей
* Pytest для тестирования
* Pydantic для валидации моделей (Request / Response)
* Enum для описания ошибок
* Allure отчет о тестировании
