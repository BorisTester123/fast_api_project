# FastAPI Books API
Проект на FastAPI с асинхронным подключением к реляционной базе данных PostgreSQL через SQLAlchemy, и настройка через ORM модель.

**Суть проекта:**
аутентификация пользователей
Базовая авторизация (Basic Auth) с безопасным хранением паролей через bcrypt.

**CRUD операции для книг:** создание, чтение, обновление и удаление информации о книгах.

**Автотесты**: используется фреймворк Pytest для удобного запуска тестов и проверки API.
**@pytest.mark.parametrize** - декоратор для проверки нескольких сценариев сразу (positive/negative).
Фикстуры для подготовки тестового окружения.


**Стэк**:

* FastAPI
* SQLAlchemy / PostgreSQL
* bcrypt для хеширования паролей
* Pytest для тестирования
* Pydantic для валидации моделей (Request / Response)
* Enum для описания ошибок
* Allure отчет о тестировании
