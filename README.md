Проект FastAPI с SQLAlchemy, асинхронным PostgreSQL, базовой аутентификацией и хешированием паролей bcrypt. CRUD-API для работы с книгами с аутентификацией пользователей. Написание автотестов для наших CRUD операций, использование библиотеки Pytest, Декоратор (mark.parametrize) для тестирования сразу нескольких сценариев (positive / negative), фикстуры (предусловия теста).
