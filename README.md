# FastAPI Books API

Проект для работы с API: реализован с помощью фреймворка FastAPI, с настроенным асинхронным подключением и сессией к реляционной базе данных PostgreSQL.

Создание таблицы в базе данных и описание полей выполнено с помощью SQLAlchemy ORM (DeclarativeBase). Также настроен репозиторий для работы с базой данных, чтобы все запросы выполнялись через слой доступа к данным.

---

## 📌 Описание проекта

### Суть проекта:

Аутентификация пользователей и базовая авторизация (Basic Auth) с безопасным хранением паролей через bcrypt.

Создание отдельной таблицы для хранения данных авторизации для работы с OpenAPI и CRUD-операциями:
- создание книг
- получение книг
- обновление книг
- удаление книг

---

### 🧪 Автотесты

Используется Pytest для тестирования API.

- `@pytest.mark.parametrize` — проверка нескольких сценариев (positive/negative)
- фикстуры для подготовки тестового окружения и тестовых данных

---

### 🛠 Стек технологий

- FastAPI
- SQLAlchemy / PostgreSQL
- bcrypt (хеширование паролей)
- Pytest (тестирование)
- Pydantic (валидация request/response моделей)
- Enum (описание ошибок)
- Allure (отчёты о тестировании)

## 🐳 Docker

Проект полностью контейнеризирован и запускается через Docker Compose.

### Сервисы:

- **db** — PostgreSQL база данных
- **backend** — FastAPI приложение

### Особенности:

- база данных запускается в отдельном контейнере
- backend ожидает готовность базы (healthcheck)
- все настройки передаются через `.env`
